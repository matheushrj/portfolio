
  # Landing Page para Profissional Audiovisual

  This is a code bundle for Landing Page para Profissional Audiovisual. The original project is available at https://www.figma.com/design/mMpthPvVyvOPu7otGjjQMI/Landing-Page-para-Profissional-Audiovisual.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.


## Publicar no GitHub Pages (Vite + React)

Este projeto já está configurado para funcionar no GitHub Pages usando caminhos relativos (`base: './'`) e saída de build em `build/`.

1) Instale as dependências
```bash
npm install
```

2) Gere o build estático
```bash
npm run build
```

3) Publique no GitHub Pages (cria/atualiza a branch `gh-pages`)
```bash
npm run deploy
```

No GitHub:
- Settings → Pages
- Branch: `gh-pages`
- Folder: `/ (root)`

`https://SEU_USUARIO.github.io/NOME_DO_REPOSITORIO/`
